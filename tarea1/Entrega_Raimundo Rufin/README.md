##### Descripción

El programa consta de 2 puntos invisibles que se desplazan de manera aleatoria a través del espacio y que al momento de hace contacto con uno de los bordes cambian su dirección y sentido para mantenerse dentro del espacio delimitado.

Estos puntos invisibles están conectados por una línea visible que se dibuja sobre el espacio a medida que ambos puntos de desplazan por este.

La línea puede ser cambiada de color controlando los parámetros RGB individualmente, donde las teclas "Y" "U" e "I" cambian el valor de "R" "G" y "B" respectivamente, a 255 y las teclas "H" "J" y "K" cambian sus valores a 0.

Además existen elipses en ambos puntos invisibles a las cuales se les puede cambiar su diámetro con las teclas "V" para aumentar y "B" para disminuir y con la tecla "C" se puede mantener en 0.

El fondo es fijo lo que permite que se mantenga el rastro que dejan los elementos a su paso, creando composiciones alterables mediante el uso de las funciones descritas anteriormente.

Para reiniciar el proceso de dibujo se debe realizar un "click" sobre la pantalla para volver a tener un fondo completamente negro.
