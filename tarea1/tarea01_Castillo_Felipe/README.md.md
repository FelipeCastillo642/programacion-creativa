Tarea 01
Autor: Felipe Castillo

En esta tarea se explora la ilusión óptica a través de la programación y el uso de clases. Dentro de este código se encuentra la interacción al utilizar el ratón o mousepad, accionando el botón izquierdo y derecho de estos, alternando las acciones para crear distintos resultados visuales.
Instrucciones:
Utilizar el botón izquierdo para alternar entre colores.
Utilizar el botón derecho para entrar en modo maleable.
El movimiento del ratón solo será activado al mover este mientras se acciona el botón derecho.
